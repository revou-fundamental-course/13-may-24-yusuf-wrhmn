Luas Segitiga

Rumus luas segitiga yaitu:

L = 1/2 × a × t

Di mana :
L = Luas
a = Alas Segitiga
t = Tinggi Segitiga

Keliling Segitiga

Rumus keliling segitiga yaitu:

K = a + b + c

Di mana
K = Keliling
a = Alas Segitiga
b = Sisi B
c = Sisi C
