Segitiga

Luas Segitiga

Rumus luas segitiga yaitu:

L = 1/2 × a × t

Di mana :

L = Luas

a = Alas Segitiga

t = Tinggi Segitiga

Hitung Luas Segitiga

L = 1/2 × a × t

L = 1/2 × 0 × 0

L = 0



Keliling Segitiga

Rumus keliling segitiga yaitu:

K = a × b × c

Di mana :

K = Keliling

a = Alas Segitiga

b = Sisi B

c = Sisi C

Hitung Keliling Segitiga

K = a × b × c

K = 0 × 0 × 0

K = 0
